Arrays

This page contains problems on arrays from geeks for geeks and their solutions.

Logic:-
From mathematics, we know that the sum of first N numbers is [N*(N+1)]/2. To determine the missing element, we will sum all the elements of the array and subtract this sum (actual sum) from [N*(N+1)]/2 (expected sum).
Space complexity:- O(1)
Time complexity:- O(n)

public int getMissingNumber(int a[])
{
  int rightPointer = a.length-1;
  int leftPointer = 0;
  int actualSum = 0;
  int expectedSum = 0;
  while(leftPointer<=rightPointer)
  {
    actualSum+=a[leftPointer];
    leftPointer++;
  }
  int numberOfElements = a.length;
  expectedSum = ((numberOfElements+1)*(numberOfElements+2))/2;
  return expectedSum - actualSum;
}

Logic:-
First find the sum of all the array elements. Now for each iteration of the loop, you can keep finding the left sum by simply adding the new element to the existing left sum. For finding the right sum, you can substract the current left sum from the total sum and add the current element to the difference (since the current element is common between the left sum and right sum). This way you can avoid using 2 nested for loops.
Space complexity:- O(1)
Time complexity:- O(n)

public int getEquilibriumPosition(int a[])
{
  int leftPointer = 0;
  int rightPointer = a.length-1;
  
  int totalSum = 0;
  while(leftPointer<=rightPointer)
  {
    totalSum += a[leftPointer];
    leftPointer++;
  }
  
  leftPointer = 0;
  int leftSum = 0;
  int rightSum = 0;
  
  while(leftPointer<=rightPointer)
  {
    leftSum += a[leftPointer];
    rightSum = totalSum - leftSum + a[leftPointer];
    if(leftSum == rightSum)
      return leftPointer+1;
    leftPointer++;
  }
  return -1;
}

Logic:-
In order to solve the problem with just one array traversal, start traversing the array from right keeping track of the maximum element encountering while traversing left so far. At any time during iteration, if an element is encountered, which is greater than or equal to the maximum element encountered so far, prefix it to the output string, update the maxSoFar and continue with the iteration. After all the array elements are traversed from right to left, you have a string with all the leaders.

public String getLeaders(int a[])
{
  int leftPointer = 0;
  int rightPointer = a.length-1;
  int maxSoFarFromRight = a[rightPointer];
  String output = "";
  
  while(rightPointer >= leftPointer)
  {
    if(a[rightPointer]>=maxSoFarFromRight)
    {
      maxSoFarFromRight = a[rightPointer];
      output = a[rightPointer] + " " + output;
    }
    rightPointer--;
  }
  
  return output.trim();
}
